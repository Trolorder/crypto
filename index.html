Ton mot de passe maître déverrouille l'accès depuis n'importe quel appareil — pas besoin de ressaisir le token GitHub. Indique juste où se trouve ton dépôt.
Le token GitHub est chiffré (AES-256) avec ton mot de passe maître. Il est sauvegardé à la fois sur cet appareil et dans ton dépôt GitHub (fichier data/vault.json, jamais en clair) — c'est ce qui permet à un seul mot de passe de fonctionner sur tous tes appareils. Le dépôt doit être public pour que « Nouvel appareil » et la lecture seule fonctionnent.
